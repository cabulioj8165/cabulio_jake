Notes on Web Programming:

Editing:

Websites are simple text files like python. Web browsers are programmers compilers, unlike python.
In order to make a website, all that is needed is a text file. Web programmers are able to use IDE
(Integrates Development Environment) but it is a lot easier to use text files like Notepad++.

Languages:

There are two main languages in making websites: XHTML and CSS. XHTML resolves around tags
and the first tag creates on a website determines the doctype.

Tags:

Almost all tags used in web programming involves opening and closing tags. Things outside tags are related
to a different object to a thing inside a tag. There are many tags used in programming. A few examples are Head,
Body, p, strong, em,a , and hr/. Usually, the first tag used in web programming is Head. Head usually includes the main
information about the website and no person using the website can see it. It is used to store keywords for search engines,
information about the browser, the title of the website, etc. The second most used tag in web programming is Body. Body includes
the information or content of the website that the user of the site can see. This includes images, audio, paragraphs, backgrounds,
animations, etc.

Other Tags:

Paragraph:
P, or paragraph, is used to allow the user to tell the browser which text in a doocument is supposed to be in
paragraph form. Users shouldn't use separate tags for each paragraph.
Horizontal Line:
Add a horizontal line by entering hr to act as a separator.
Strong
Using strong puts the text selected in it bold.
Em
Em italicizes the text in the enclosed area.

Comments:

Comments can be added to a website like python or other program languages. Links can be added to the page by using an anchor tag.

Header:

Headers are used to determine which words on the website are bigger and more important. H1 is the biggest
and H6 is the smallest. Page navigation can be added by using links.

Tags:

Almost all tags used in web programming involves opening and closing tags. Things outside tags are related
to a different object to a thing inside a tag. There are many tags used in programming. A few examples are

  • Head
  • Body
  • p
  • strong
  • em
  • a
  • hr/

Usually, the first tag used in web programming is Head. Head usually includes the main
information about the website and no person using the website can see it. It is used to store keywords for search engines,
information about the browser, the title of the website, etc. The second most used tag in web programming is Body. Body includes
the information or content of the website that the user of the site can see. This includes images, audio, paragraphs, backgrounds,
animations, etc.

-Written by Mr. Cabulio on 3-2-17

Other tags:

  3. Strong
